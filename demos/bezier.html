Linear Bezier Curve

A linear bezier curve is a line segment defined by endpoints \( P_0 \) and \( P_1 \). Formally, it is the set of all points given by the parametric equation: $$ L(t) = (1 - t) P_0 + t P_1, 0 \le t \le 1 $$
